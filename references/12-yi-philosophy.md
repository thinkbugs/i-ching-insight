## 一、形而上

### 1.1 太极

**定义**：
- 宇宙万物的本源
- 阴阳未分之前的状态
- 无形无象，无始无终

**特征**：
- **无形**：不可见，不可感知
- **无极**：无边无际，无穷无尽
- **一理**：宇宙万物的根本原理
- **动静**：静为无极，动为太极

**经典表述**：
- "易有太极，是生两仪"（《系辞上》）
- "太极者，无极而生，动静之机，阴阳之母也"（周敦颐）
- "无极而太极"（朱熹）

### 1.2 无极

**定义**：
- 比太极更原始的状态
- 彻底的虚无，绝对的寂静
- 道家的"道"

**特征**：
- **无**：无物，无象，无名
- **静**：绝对静止，无任何变化
- **本**：一切存在的根源

**与太极的关系**：
- 无极是太极之前的状态
- 太极由无极而生
- "无极而太极"

### 1.3 道

**定义**：
- 宇宙万物的总规律
- 最高的哲学范畴
- 不可言说的终极实在

**特征**：
- **不可言说**："道可道，非常道"
- **无形无象**："视之不见，听之不闻"
- **生生不息**："道生一，一生二，二生三，三生万物"

**与太极的关系**：
- 道是太极的根本
- 太极是道的显现
- 道体太极用

## 二、形而下

### 2.1 阴阳

**定义**：
- 宇宙万物最基本的对立统一范畴
- 气的两种存在形式
- 万物运动变化的内在动力

**特征**：
- **对立**：阴阳相反，相互排斥
- **统一**：阴阳相互依存，不可分割
- **消长**：阴阳此消彼长，动态平衡
- **转化**：在一定条件下，阴阳相互转化

**阴阳的属性**：
- 阳：刚健、主动、上升、明亮、温热
- 阴：柔顺、被动、下降、阴暗、寒冷

**阴阳的关系**：
- **阴阳互根**：阴阳相互依存
- **阴阳消长**：阴阳此消彼长
- **阴阳转化**：阴极生阳，阳极生阴
- **阴阳平衡**：阴阳相对平衡，万物生长

### 2.2 四象

**定义**：
- 阴阳的进一步分化
- 四种基本状态
- 四季、四方、四时的象征

**四象的构成**：
- **太阳**（老阳）：阳中之阳，至阳
- **少阴**：阳中之阴
- **少阳**：阴中之阳
- **太阴**（老阴）：阴中之阴，至阴

**四象的象征**：
- 四季：春（少阳）、夏（太阳）、秋（少阴）、冬（太阴）
- 四方：东（少阳）、南（太阳）、西（少阴）、北（太阴）
- 四时：晨（少阳）、午（太阳）、昏（少阴）、夜（太阴）

### 2.3 八卦

**定义**：
- 阴阳的三重组合
- 八种基本卦象
- 宇宙万物的八种基本模式

**八卦的构成**：
- **乾**（☰）：三阳，天，刚健
- **坤**（☷）：三阴，地，柔顺
- **震**（☳）：上二阴下一阳，雷，震动
- **巽**（☴）：上二阳下一阴，风，顺从
- **坎**（☵）：上下中阴，水，险陷
- **离**（☲）：上下中阳，火，光明
- **艮**（☶）：上一下二阳，山，静止
- **兑**（☱）：上一下二阴，泽，喜悦

**八卦的象征**：
- 乾天坤地：天地定位
- 震雷巽风：雷风相薄
- 坎水离火：水火不相射
- 艮山兑泽：山泽通气

### 2.4 六十四卦

**定义**：
- 八卦的两两组合
- 六十四种复杂卦象
- 宇宙万物的六十四种状态

**六十四卦的构成**：
- 六十四卦 = 8 × 8
- 每卦六爻，由上下两卦组成
- 初二爻为地，三四爻为人，五上爻为天

**六十四卦的象征**：
- 每卦代表一种特定的状态或情境
- 反映宇宙万物的复杂变化
- 提供应对各种情境的智慧

## 三、太极阴阳

### 3.1 太极生两仪

**表述**：
"易有太极，是生两仪"（《系辞上》）

**解释**：
- 太极是宇宙万物的本源
- 太极一动，分化为阴阳两仪
- 阴阳是宇宙万物最根本的对立统一

**图示**：
```
        太极
         |
    +----+----+
    |         |
   阴        阳
  (— —)    (——)
```

### 3.2 两仪生四象

**表述**：
"两仪生四象"

**解释**：
- 阴阳进一步分化为四象
- 四象是阴阳的不同组合状态
- 四象反映事物的多样性和层次性

**图示**：
```
          太极
           |
      +----+----+
      |         |
     阴        阳
      |         |
   +--+--+   +--+--+
   |  |  |   |  |  |
  太 少 少 太 太 少 少 太
  阴 阴 阳 阳 阴 阳 阳 阴
```

### 3.3 四象生八卦

**表述**：
"四象生八卦"

**解释**：
- 四象进一步组合为八卦
- 八卦是阴阳的三重组合
- 八卦是宇宙万物的八种基本模式

**图示**：
```
                太极
                 |
             +---+---+
             |       |
            阴       阳
             |       |
          +--+--+   +--+--+
          |  |  |   |  |  |
         太 少 少 太 太 少 少 太
         阴 阴 阳 阳 阴 阳 阳 阴
          |  |  |   |  |  |  |  |
         坤 艮 坎 巽 震 离 兑 乾
```

### 3.4 八卦生万物

**表述**：
"八卦生万物"

**解释**：
- 八卦两两组合为六十四卦
- 六十四卦代表宇宙万物的各种状态
- 万物都可以用六十四卦来描述和解释

**原理**：
- 万物皆由阴阳构成
- 阴阳的不同组合形成万物
- 六十四卦是阴阳组合的全部可能性

## 四、三才之道

### 4.1 天道

**定义**：
- 天的运行规律
- 阳刚之道的体现
- 代表刚健、创造、主导

**特征**：
- **刚健**："天行健"
- **生生不息**："生生之谓易"
- **刚而不强**：刚健而有节制

**天道的内容**：
- 日月星辰的运行规律
- 四季气候的变化规律
- 宇宙万物的生成规律

**天道的象征**：
- 乾卦代表天道
- "天行健，君子以自强不息"
- 君子效法天道，刚健有为

### 4.2 地道

**定义**：
- 地的运行规律
- 阴柔之道的体现
- 代表柔顺、承载、顺从

**特征**：
- **柔顺**："地势坤"
- **厚德载物**：以深厚的德行承载万物
- **顺而不从**：柔顺而有原则

**地道的内容**：
- 山川河流的分布规律
- 万物生长的承载规律
- 资源供给的分配规律

**地道的象征**：
- 坤卦代表地道
- "地势坤，君子以厚德载物"
- 君子效法地道，柔顺包容

### 4.3 人道

**定义**：
- 人的行为规律
- 天人合一的体现
- 代表中庸、和谐、中正

**特征**：
- **中庸**：不偏不倚，恰到好处
- **和谐**：天人和谐，人际和谐
- **中正**：端正中正，不偏不倚

**人道的内容**：
- 道德修养的规律
- 社会交往的规律
- 个人发展的规律

**人道的象征**：
- 人居于天地之间
- 效法天地之道
- "与天地参"（与天地并立为三）

### 4.4 三才统一

**表述**：
"立天之道，曰阴与阳；立地之道，曰柔与刚；立人之道，曰仁与义"（《说卦传》）

**解释**：
- 天道：阴阳
- 地道：柔刚
- 人道：仁义

**三才的关系**：
- 天道统摄地道
- 地道承载人道
- 人道效法天地

**三才的统一**：
- "天人合一"
- "万物与我为一"
- 人与天地万物是一个整体

**六爻对应三才**：
- 初二爻：地道
- 三四爻：人道
- 五上爻：天道

## 易经哲学的核心思想

### 1. 生生不息

**表述**："生生之谓易"

**含义**：
- 宇宙万物不断生成、发展、变化
- 变化是宇宙的根本特征
- 永远处于动态之中

### 2. 阴阳平衡

**表述**："一阴一阳之谓道"

**含义**：
- 阴阳相互依存，相互制约
- 阴阳平衡是万物正常发展的基础
- 阴阳失调会导致问题

### 3. 变通之道

**表述**："穷则变，变则通，通则久"

**含义**：
- 困境中要寻求变化
- 变化之后才能通达
- 通达才能长久

### 4. 中庸之道

**表述**："中正以观天下"

**含义**：
- 不偏不倚，恰到好处
- 端正中正，不偏不倚
- 中庸是最高的智慧

### 5. 天人合一

**表述**："与天地参"

**含义**：
- 人与天地万物是一个整体
- 人效法天地之道
- 人与天地和谐共生
