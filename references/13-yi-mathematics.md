## 一、二进制与卦象

### 1.1 莱布尼茨的发现

**历史背景**：
- 莱布尼茨（1646-1716）：德国数学家、哲学家
- 17世纪末，莱布尼茨发明二进制
- 18世纪初，莱布尼茨接触到易经卦象
- 发现卦象与二进制的对应关系

**经典表述**：
- "二进制是易经的数学表达"
- "八卦是二进制的早期发现"

### 1.2 二进制对应关系

**阴阳爻的二进制表示**：
- 阳爻（—）：1
- 阴爻（— —）：0

**八卦的二进制表示**：
| 八卦 | 符号 | 二进制 | 十进制 |
|------|------|--------|--------|
| 乾 | ☰ | 111 | 7 |
| 兑 | ☱ | 011 | 3 |
| 离 | ☲ | 101 | 5 |
| 震 | ☳ | 001 | 1 |
| 巽 | ☴ | 110 | 6 |
| 坎 | ☵ | 010 | 2 |
| 艮 | ☶ | 100 | 4 |
| 坤 | ☷ | 000 | 0 |

**六十四卦的二进制表示**：
- 每卦六爻，对应6位二进制数
- 例如：乾卦（☰☰）= 111111（十进制63）
- 例如：坤卦（☷☷）= 000000（十进制0）

### 1.3 卦象的二进制表示

**上卦与下卦**：
- 上卦：第4、5、6爻（高位）
- 下卦：第1、2、3爻（低位）

**计算方法**：
- 例如：泰卦（☰☷）
  - 上卦（乾）= 111 = 7
  - 下卦（坤）= 000 = 0
  - 泰卦 = 111000 = 56（十进制）

**卦象的数学意义**：
- 二进制表示了卦象的数学结构
- 可以用数学方法研究卦象的性质
- 为计算机处理易经提供基础

## 二、64卦对称群

### 2.1 对称性概念

**群论基础**：
- 群：满足封闭性、结合律、单位元、逆元的代数结构
- 对称性：保持某些性质不变的变换
- 对称群：所有对称变换的集合

**卦象的对称性**：
- 错卦：阴阳全反
- 综卦：上下反转
- 互卦：取中间四爻

### 2.2 错卦对称

**定义**：
- 错卦：阴阳爻全部相反的卦
- 例如：乾卦（☰☰）的错卦是坤卦（☷☷）

**数学表示**：
- 错卦 = ~卦象（按位取反）
- 例如：111111 → 000000

**错卦的性质**：
- 错卦的错卦是原卦（自反性）
- 每卦都有唯一的错卦
- 错卦构成32对

### 2.3 综卦对称

**定义**：
- 综卦：上下卦反转的卦
- 例如：泰卦（☰☷）的综卦是否卦（☷☰）

**数学表示**：
- 综卦 = reverse(卦象)（位序反转）
- 例如：111000 → 000111

**综卦的性质**：
- 综卦的综卦是原卦（自反性）
- 有些卦的综卦是自身（自综卦）
- 自综卦：乾、坤、坎、离、大过、小过、中孚、颐（8卦）

### 2.4 互卦对称

**定义**：
- 互卦：取本卦的2、3、4爻为下互，3、4、5爻为上互
- 例如：泰卦（☰☷）的互卦是归妹卦（☱☳）

**数学表示**：
- 上互 = 卦象[2:5]（第3、4、5爻）
- 下互 = 卦象[1:4]（第2、3、4爻）
- 互卦 = 上互 + 下互

**互卦的性质**：
- 互卦反映本卦的内部结构
- 用于深入分析本卦的含义

## 三、卦序数理

### 3.1 卦序排列规律

**卦序的概念**：
- 六十四卦的排列顺序
- 反映某种数学或哲学规律
- 主要有两种卦序：文王卦序和伏羲卦序

### 3.2 文王卦序

**特点**：
- 上经30卦，下经34卦
- 反映社会历史发展的规律
- 哲学意义强于数学意义

**上经（1-30卦）**：
- 乾（☰☰）、坤（☷☷）：天地定位
- 屯（☵☳）、蒙（☳☵）：开天辟地
- 需（☰☵）、讼（☵☰）：生存斗争
- ...至离（☲☲）

**下经（31-64卦）**：
- 咸（☱☶）、恒（☶☴）：人伦之始
- 遯（☲☶）、大壮（☳☰）：进退之理
- ...至未济（☲☵）

**数学规律**：
- 卦序对（错卦或综卦）
- 某些卦序呈现对称性

### 3.3 伏羲卦序

**特点**：
- 按照二进制数值排列
- 数学规律强
- 反映宇宙生成顺序

**伏羲六十四卦方圆图**：
- 圆图：按逆时针排列
- 方图：按矩阵排列

**圆图排列**：
- 从乾（☰☰）开始，逆时针排列
- 反映阴阳消长规律

**方图排列**：
- 按照八卦组合排列
- 反映宇宙结构

### 3.4 卦序的数学意义

**信息论视角**：
- 六十四卦 = 6位二进制 = 64种状态
- 信息量 = log₂(64) = 6比特
- 卦象是信息的编码方式

**群论视角**：
- 六十四卦构成一个群
- 对称变换（错卦、综卦）是群元素
- 群的结构反映卦象的对称性

**数论视角**：
- 卦序与数字的关系
- 卦象的代数运算
- 卦象的数理性质

## 四、数理分析

### 4.1 卦象的代数结构

**代数系统**：
- 六十四卦集合 G = {0, 1, 2, ..., 63}
- 运算：
  - 按位取反（错卦）：~a
  - 按位反转（综卦）：reverse(a)
  - 按位异或：a ⊕ b

**群结构**：
- (G, ⊕) 是一个群
- 单位元：0（坤卦）
- 逆元：自身（a ⊕ a = 0）
- 结合律：a ⊕ (b ⊕ c) = (a ⊕ b) ⊕ c

### 4.2 群论视角

**对称群**：
- 错卦对称群：C₂群（两个元素）
- 综卦对称群：C₂群（两个元素）
- 组合对称群：D₄群（八个元素）

**不变性**：
- 错卦变换下，某些性质不变
- 综卦变换下，某些性质不变
- 组合变换下，某些性质不变

### 4.3 信息论视角

**信息编码**：
- 卦象是信息的编码
- 每爻携带1比特信息
- 每卦携带6比特信息

**熵**：
- 六十四卦的熵 = log₂(64) = 6比特
- 反映了卦象的信息量

**压缩**：
- 卦象可以压缩
- 重复卦象可以用符号表示
- 象征了信息的压缩和编码
